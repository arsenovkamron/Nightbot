# Telegram Giveaway Bot

Бот для розыгрышей в Telegram-канале с живым таймером, кнопкой участия и автоматическим выбором победителей.

## Что умеет

- Запускать розыгрыш по команде администратора
- Публиковать пост с фото и описанием
- Обновлять таймер и количество участников прямо в сообщении
- Не допускать повторное участие одного и того же пользователя
- Проверять подписку на канал перед участием
- Автоматически выбирать победителей после завершения
- Восстанавливаться после перезапуска, потому что активный розыгрыш хранится в SQLite

## Переменные окружения

Перед запуском нужно задать:

```powershell
$env:BOT_TOKEN="your_bot_token"
$env:ADMIN_ID="123456789"
$env:CHANNEL_ID="-1001234567890"
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Команды

- `/start` - приветствие
- `/panel` - панель администратора
- `/status` - статус бота
- `/start_giveaway <секунды> <победители>` - создать розыгрыш
- `/cancel_giveaway` - удалить черновик розыгрыша

Пример:

```text
/start_giveaway 3600 3
```

## Как проходит создание розыгрыша

1. Админ отправляет `/start_giveaway 3600 3`
2. Бот просит фото
3. Бот просит текст
4. Бот публикует пост в канале
5. Пользователи нажимают кнопку участия
6. Таймер и число участников обновляются автоматически
7. После завершения бот публикует победителей

## Примечания

- Бот должен быть администратором канала
- Для проверки подписки бот должен иметь доступ к участникам канала
- Таймер обновляется каждые 5 секунд, чтобы не упираться в ограничения Telegram API
